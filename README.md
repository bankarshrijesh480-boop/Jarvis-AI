# 🤖 JARVIS AI

> **Your Personal AI Operating System**

A modern AI-powered workspace inspired by **ChatGPT, Claude, Perplexity, and Iron Man's J.A.R.V.I.S.**

JARVIS AI is an intelligent assistant built with **React, Firebase, and modern AI APIs**, designed to provide conversations, code generation, image creation, voice interaction, file analysis, and much more—all inside one beautiful interface.

# ⚙️ Installation

Clone the repository

```bash
git clone https://github.com/YOUR_USERNAME/Jarvis-AI.git
```

Go to the project

```bash
cd Jarvis-AI
```

Install dependencies

```bash
npm install
```

Run development server

```bash
npm run dev
```
